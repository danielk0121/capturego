## 작업 내용
- 현재: 웹앱 설정 페이지 > 현황, 라이센스 > 2개 항목이 설정 메인 페이지에 있음
- 따라서, 설정 페이지 화면이 세로로 길어지는 안좋은 효과
- 라이센스 강요하는 안좋은 효과
- 라이센스 키 등록하는 별도 페이지 추가 필요
- 우상단 언어 변경 버튼 옆에 추가
- 현재 : 메뉴 트레이 > 개발자 응원 > 깃헙 페이지로 이동
- 수정 : 메뉴 트레이 > 이 앱에 대하여(about) > 깃헙 페이지로 이동
- 메뉴 트레이 > 라이센스 키 등록 > 새로운 메뉴 추가 > 라이센스 키 등록 웹앱 페이지로 이동
