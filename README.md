# capturego

## Overview

A Go-based project.

## Prerequisites

- Go 1.21+

## Getting Started

```bash
# Clone the repository
git clone https://github.com/danielk0121/capturego.git
cd capturego

# Install dependencies
go mod download

# Build
go build ./...

# Run tests
go test ./...
```
